Flipkart Sales Analytics Dashboard | Power BI

This project features an interactive Power BI dashboard built using Flipkart sales data to analyze and visualize key business metrics. 
The dashboard provides actionable insights into sales performance, revenue trends, product categories, customer behavior, and regional distribution.

Key Features

1) Sales & Revenue Analysis – Track overall sales performance and growth trends
2) Category & Product Insights – Identify top-performing and underperforming product categories
3) Regional Performance – Analyze sales distribution across locations
4) Customer Insights – Understand buying patterns and order behavior
5) Interactive Filters & Drill-Downs – Dynamic slicing by time, category, and region

Tools & Technologies

Power BI
DAX
Power Query
Data Modeling & Visualization

Objective

To demonstrate strong data analysis, visualization, and storytelling skills by converting raw e-commerce data into meaningful business insights for decision-making.
